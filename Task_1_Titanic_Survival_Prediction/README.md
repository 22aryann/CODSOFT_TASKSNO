# Task 1 - Titanic Survival Prediction

This project uses the Titanic dataset to build a machine learning model that predicts whether a passenger survived or not.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors

## Project Workflow
- Data Understanding
- Data Cleaning
- Exploratory Data Analysis
- Data Preprocessing
- Model Training
- Model Comparison
- Hyperparameter Tuning
- Model Evaluation
- Survival Prediction
