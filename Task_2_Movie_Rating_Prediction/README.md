# Movie Rating Prediction with Python

## CodSoft Data Science Internship - Task 2

This project analyzes an IMDb Indian Movies dataset and builds a machine learning model to predict movie ratings.

## Project Objectives

- Analyze historical movie data
- Find the year with the best average movie ratings
- Analyze the relationship between movie duration and rating
- Find the top-rated movies overall and by year
- Analyze movie popularity based on votes
- Identify directors with the most movies
- Identify actors with the most movie appearances
- Discover additional trends in the movie dataset
- Build a machine learning model to predict movie ratings

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning

A regression-based machine learning model is used to predict IMDb movie ratings using features such as:

- Genre
- Director
- Actors
- Year
- Duration
- Votes

## Dataset

IMDb Movies India Dataset
