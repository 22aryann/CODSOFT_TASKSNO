# CODSOFT Data Science Internship Tasks

This repository contains the Data Science projects completed as part of my internship at CODSOFT.

The repository includes three machine learning projects covering data preprocessing, exploratory data analysis, visualization, model training, evaluation, and prediction.

## 📌 Tasks Completed

### Task 1: Titanic Survival Prediction

Built a machine learning classification model to predict whether a passenger on the Titanic survived based on features such as age, gender, passenger class, and fare.

**Key Steps:**
- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Data Visualization
- Training Multiple Classification Models
- Model Comparison
- Hyperparameter Tuning
- Model Evaluation
- Survival Prediction

**Technologies:** Python, Pandas, NumPy, Matplotlib, Scikit-learn

---

### Task 2: Movie Rating Prediction

Developed a machine learning regression model to predict movie ratings based on various features from the Indian movies dataset.

**Key Steps:**
- Data Understanding and Cleaning
- Handling Missing Values
- Data Preprocessing
- Feature Engineering
- Exploratory Data Analysis
- Model Training
- Model Evaluation
- Movie Rating Prediction

**Technologies:** Python, Pandas, NumPy, Matplotlib, Scikit-learn

---

### Task 3: Iris Flower Classification

Built a machine learning classification model to classify Iris flowers into Setosa, Versicolor, and Virginica based on their sepal and petal measurements.

**Key Steps:**
- Data Exploration
- Data Cleaning
- Data Visualization
- Feature Encoding and Scaling
- Training Multiple Classification Models
- Model Performance Comparison
- Confusion Matrix Analysis
- Feature Importance Analysis
- Iris Species Prediction

**Technologies:** Python, Pandas, NumPy, Matplotlib, Scikit-learn
