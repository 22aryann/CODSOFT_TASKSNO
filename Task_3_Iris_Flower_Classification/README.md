# Task 3 - Iris Flower Classification

## Overview

This project is part of my Data Science Internship at CODSOFT.

The objective of this project is to build a machine learning model that can classify Iris flowers into different species based on their sepal and petal measurements.

The three Iris species included in the dataset are:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Dataset

The Iris dataset contains the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

## Project Workflow

1. Importing required libraries
2. Loading and understanding the dataset
3. Checking missing values and duplicate records
4. Exploratory Data Analysis and visualization
5. Encoding the target variable
6. Splitting the dataset into training and testing sets
7. Feature scaling
8. Training multiple classification models
9. Comparing model performance
10. Evaluating the final model
11. Analyzing feature importance
12. Predicting the species of a new Iris flower

## Machine Learning Models Used

The following classification algorithms were trained and compared:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Model Evaluation

The models were evaluated using:

- Accuracy Score
- Classification Report
- Confusion Matrix

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Conclusion

The machine learning models successfully classified Iris flowers based on their sepal and petal measurements. Multiple classification algorithms were compared to evaluate their performance. The project demonstrates the complete machine learning workflow, including data preprocessing, visualization, model training, evaluation, and prediction.
